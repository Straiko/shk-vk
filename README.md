# SHK Bot (VK)

Бот ВКонтакте для сканирования и генерации штрих-кодов и QR-кодов.

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Настроить токен
cp .env.example .env
# Вписать VK_BOT_TOKEN из vk.com

# 3. Запустить
python bot.py
```

## Настройка VK бота

1. Перейди в vk.com → Сообщества → Создать сообщество
2. Выбери "Сообщество для ботов"
3. Настройки → API → Длинный запрос (Long Poll)
4. Включи Long Poll API
5. Скопируй токен доступа

## Деплой на bot-host.ru

1. Зарегистрируйся на bothost.ru
2. Создай новый проект
3. Загрузи код через GitHub
4. В переменных окружения укажи VK_BOT_TOKEN
5. Запусти
